#  Proyecto Deportes UNAL

Sistema para gestionar comunidades deportivas de la Universidad Nacional.

##  Descripción

El sistema permite registrar estudiantes con los deportes que practican y los que les interesan, identificar comunidades deportivas (estudiantes que comparten deportes) y buscar conexiones hacia deportes de interés.

---

##  Estructuras de Datos Implementadas

| Estructura | Uso | Complejidad |
|------------|-----|-------------|
| Árbol AVL | Almacenamiento de estudiantes por ID | O(log n) |
| Listas Enlazadas | Deportes que practica y de interés por estudiante | O(1) |
| Matriz de Adyacencia | Conexiones entre estudiantes | O(1) |
